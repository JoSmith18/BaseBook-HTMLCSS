Other Wall

This page should show the user the posts from another users wall.

The page should have:

  • A header that tells the user whose page they are on.
  • A link to the the page where they can post a comment (ex. /wall/5/comment)

  • The posts for this users wall. Each post should

    • contain a link to the author's wall
    • show the posts message

To do this, the page will be given a dictionary named user that looks like this:

{
    'username':
    'natec425',
    'id':
    1,
    'wall': [{
        'author_id': 2,
        'author': 'mmclark',
        'timestamp': pendulum.now(),
        'message': 'Hey There'
    }]
}
